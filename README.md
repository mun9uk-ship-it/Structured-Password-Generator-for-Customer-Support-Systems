<h1>🔐 Smart Password Generator (Python CLI Tool)</h1>

<h2>🚀 Overview</h2>
<p>
This is a simple yet powerful Python CLI tool designed to help customer support teams
generate structured and easy-to-remember passwords using a practical formula.
</p>

<h2>🧠 How It Works</h2>
<ul>
<li><b>Step A:</b> Sum the first 5 digits of the phone number</li>
<li><b>Step B:</b> Use the 3rd character of the project name (Upper + Lower)</li>
<li><b>Step C:</b> Apply modulus between the last 2 digits</li>
</ul>

<h2>✅ Features</h2>
<ul>
<li>✔ Input validation</li>
<li>✔ Structured password generation</li>
<li>✔ Multiple formula outputs</li>
<li>✔ Beginner-friendly code</li>
<li>✔ No external libraries required ✅</li>
</ul>

<h2>💡 Real-Life Benefits</h2>
<ul>
<li>✅ Faster workflow for customer support teams</li>
<li>✅ Easy-to-remember passwords</li>
<li>✅ No need to store or write down passwords</li>
<li>✅ Consistent and repeatable password system</li>
</ul>

<p><b>⭐ Powerful Advantage:</b></p>
<p>
Even if you are away from your customer support desk, or your computer is not working,
you can still remember the formula and manually regenerate the password using only:
</p>
<ul>
<li>📱 Phone number</li>
<li>🔤 Project name</li>
</ul>

<p>
This makes the system reliable, portable, and independent from any device.
</p>
