<h1>🔐 Smart Password Generator (Python CLI Tool)</h1>

<h2>🚀 Overview</h2>
<p>
This is a simple yet powerful Python CLI tool designed to help customer support teams
generate structured and easy-to-remember passwords using a practical formula.
</p>

<h2>🧠 How It Works</h2>
<ul>
<li><b>Step A:</b> Sum the first 5 digits of the phone number</li>
<li><b>Step B:</b> Use the 3rd character of the project name (Upper + Lower)</li>
<li><b>Step C:</b> Apply modulus between the last 2 digits</li>
</ul>
